# addコマンドについて

* gitで変更管理するファイルとして追加するコマンド
  * ディレクトリに追加するだけではgit上で管理することにならない
  * どのファイルを追加するかは任意で選択できる


# commitコマンドについて
* ファイルの変更や追加をリポジトリに保存するコマンド。

* 文法
``` git commit
``` git commit -m 'commit comment'
